**AmarShop** is a high-fidelity, production-ready luxury eCommerce platform designed for a premium shopping experience. Built with the latest frontend technologies, it focuses on high-performance rendering, stunning micro-animations, and a "Command Center" dashboard for elite users.

---

## ✨ Core Features

### 🚀 Performance & UI/UX
- **Modern Tech Stack**: Powered by **React 19** and **Vite 6** for near-instant load times and HMR.
- **Tailwind CSS v4**: Utilizing the latest CSS-first engine for a sleek, performant design system.
- **Glassmorphism Design**: High-fidelity UI using translucent layers, soft shadows, and elite typography.
- **Fluid Animations**: Smooth transitions powered by **Motion v12**, ensuring a "living" interface.

### 🛍️ Shopping Experience
- **Flash Deals**: Time-sensitive sales with real-time countdowns and stock progress indicators.
- **Intelligent Search**: Dynamic product filtering and category-based navigation.
- **Wishlist & Cart**: Persistent shopping state with smooth slide-out drawers.
- **Product Tracking**: Advanced order tracking UI with real-time status visualization.

### 👤 User Command Center
- **Elite Dashboard**: A centralized portal for managing orders, addresses, and account preferences.
- **Membership Levels**: Visual cues and perks for "Elite Members" and loyal shoppers.
- **Activity History**: Detailed order archives with status-based styling and quick-action links.

## 🚀 Getting Started

### Prerequisites
- **Node.js** (v18.x or higher)
- **npm** or **pnpm**

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sherazi-IT-Ltd/Ecommerce-UI-Design-1.git
   cd Ecommerce-UI-Design-1
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Launch Development Server:**
   ```bash
   npm run dev
   ```

4. **Production Build:**
   ```bash
   npm run build
   ```
